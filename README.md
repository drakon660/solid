# SOLID

This repository contains only the Liskov Substitution Principle (LSP) from the SOLID principles.

## Documents

The `Documents` folder contains 3 original papers:

- Barbara Liskov - "Data Abstraction and Hierarchy" (OOPSLA)
- 2 documents by Barabara Liskov and Jeannette Wing

## LiskovSamples

The `LiskovSamples` folder contains .NET C# LSP code samples ranging from simple to complex.
